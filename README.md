# ESP32-CAM Remote Control Engineering Vehicle

This project implements a remote-controlled engineering vehicle (excavator/forklift) with real-time video feedback.

## Project Structure

- **firmware/**: ESP32-CAM firmware (Arduino/PlatformIO).
- **mobile-app/**: Flutter mobile application for control and video display.
- **server/**: Node.js Relay Server for remote control (Phase 2).
- **docs/**: Project documentation and plans.

## Getting Started

Please verify the hardware in `docs/implementation_plan.md` and follow `docs/task.md` for progress.
